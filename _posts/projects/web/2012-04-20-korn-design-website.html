Korn Design is a brand strategy and design firm with two offices in Boston and New York City. I was contracted by PixelSlam to create a responsive site design built on top of the WordPress platform. The site features bold, colorful imagery to highlight an extensive portfolio.

The site uses LESS to handle CSS preprocessing for a custom responsive WordPress theme. The site makes heavy use of WordPress Custom Post Types to make organizing content easy. The masonry-style layout on the portfolio page is created using the Isotope jQuery plugin from Metafizzy.
