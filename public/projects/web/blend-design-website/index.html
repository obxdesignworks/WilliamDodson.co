Blend Design Website

Customer communication and design services

Description:

Blend Design and Interactive Studio provides customer communication and design services. They contacted me by recommendation to help them build their business website based on wireframes for three distinct page templates.

The site uses Sass, Bourbon, and Neat to handle CSS preprocessing for the custom responsive WordPress theme. The site uses WordPress Custom Post Types to help make organizing content easy. The masonry-style layout on the portfolio page is created using the Isotope jQuery plugin from Metafizzy.
