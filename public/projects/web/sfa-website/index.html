Science Festival Alliance Website

A professional community dedicated to provide better science and technology festivals

Description:

The Science Festival Alliance is an MIT initiative whose goal is to foster a professional community dedicated to provide better science and technology festivals. I was contracted by PixelSlam to to create a custom site design powered by WordPress. The site features a custom events calendar to help people find science festivals all over the country.

The site uses Sass and Compass to handle CSS preprocessing for the custom WordPress theme. The site uses WordPress Custom Post Types to help make organizing content easy. I also developed a custom Events widget which can be easily embedded on any external site to allow SFA to share their events anywhere.
